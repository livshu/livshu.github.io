Personal Finance Tracker

A personal Django web app for personal and joint finances. I have little front-end experience so I have been using this project as a learning tool. Born from the frustration of existing apps not handling shared expenses or an overview of investments properly!

Features

Bank-specific Import
Bank-specific parsers for Lloyds, Santander, Amex with upload, preview, confirm workflow.
Auto-Categorisation
Rule-based merchant matching assigns categories automatically on import.
Dashboard
Chart.js visualisation: spending by category, monthly trends, income vs expenditure.
Drilldowns
Click any category to see individual transactions grouped by category with month filtering.

Architecture

Django structure: single core app with Account, Category, and Transaction models. Bank-specific CSV parsing isolated from the shared import flow.

Roadmap

Duplicate detection, merchant normalisation, recurring transaction support, savings goals, and budgeting features. The project is actively maintained and evolving as my personal finances do so too!
