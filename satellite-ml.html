Can unsupervised ML read a disaster from space in real time?

K-means clustering applied to Landsat 8 satellite imagery of Nicaragua's 2015 Momotombo volcanic eruption, using PCA and ICA preprocessing to identify landscape features without any labelled training data.

The problem

When Nicaragua's Momotombo volcano erupted in 2015, satellites captured it across seven light wavelengths: visible colour, infrared, and thermal heat. The question: could a machine learning algorithm, given nothing but those raw pixel values, automatically figure out what part of the landscape it's looking at?

Method

K-means clustering was tested across three pipelines: raw data, data preprocessed with PCA, and data preprocessed with ICA. The thermal band (the key signal for detecting fresh hot lava) was so different in scale from the other bands that raw data alone couldn't pick it out, even when its weighting was doubled. Preprocessing is essential!

PCA reduces correlated signals into uncorrelated components, cleaning up redundancy. ICA goes further, separating genuinely independent sources - a better fit for satellite spectral data, which rarely follows the neat Gaussian assumptions simpler methods rely on.

ICA successfully identified all four key landscape features using just 5 clusters. PCA needed 6. Raw data never found the new lava flow at all.

Results & limitations

Silhouette scores confirmed acceptable clustering across all models, with 6-cluster PCA scoring highest at 0.447. However, qualitative comparison with the original image proved more informative than the metric alone — a reminder that human interpretation of ML outputs still matters.

One honest limitation: cloud shadows consistently fooled the model into labelling dark patches as old lava flows, overstating the extent of burnt terrain. Environmental artefacts like shadow, cloud cover, and sensor angle must be accounted for in any real deployment.

Why?

Real-time segmentation of natural disasters (fires, floods, volcanic events) from satellite data could save lives if it can be automated. This work shows that unsupervised ML is a promising approach, requiring no pre-labelled training data which rarely exists in the immediate aftermath of a crisis.
